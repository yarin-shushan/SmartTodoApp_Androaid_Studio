# 📱 SmartToDoApp

> **"אדם בחייו אין לו זמן שיהיה לו זמן לכול" -- יהודה עמיחי**

                                                                                                                                                                                                                     SmartToDoApp

היא אפליקציית ניהול משימות חכמה שנועדה לאפשר למשתמש לנהל את
זמנו בצורה יעילה באמצעות תיעדוף משימות, סינון חכם וניתוח עומס.\
האפליקציה מרכזת את כל המטלות במקום אחד ומציגה אותן לפי חשיבות, דחיפות
ותאריך יעד --- עם ממשק מודרני, ברור ונגיש.

------------------------------------------------------------------------

## ✨ פיצ'רים מרכזיים

1.  הרשמה/התחברות + שמירת משתמש (Firebase Auth)
2.  יצירה, עריכה, מחיקה וארכוב של משימות (Firestore)
3.  מאפייני משימה: כותרת, תיאור, קטגוריה, עדיפות, תאריך יעד וחיווי
    איחור\
4.  חיפוש, מיון וסינון לפי קטגוריה/תאריך/עדיפות
5.  מסך סטטיסטיקות עם גרפי עוגה (הושלמו/ממתינות/באיחור)
6.  תצוגת לוח שנה + סינון משימות לפי יום
7.  מצב כהה, תמיכה ב־RTL, ועיצוב Material Design 3
8.  ממשק רספונסיבי: RecyclerView, AutoSizeText, ConstraintLayout

------------------------------------------------------------------------

# 🛠️ טכנולוגיות חיצוניות וספריות מיוחדות בפרויקט

## 🔥 Firebase 

-   Firebase Authentication - ניהול משתמשים 
-   Cloud Firestore - בסיס נתונים בענן לניהול משימות
-   Firebase Analytics - מעקב שימוש
-   Firebase BoM - ניהול מרכזי לגרסאות Firebase
-   Google Services Plugin - חיבור האפליקציה ל-Firebase

## 🌐 Networking / API 

-   Retrofit 2 - לקוח API לביצוע בקשות רשת
-   Gson Converter - המרת JSON לאובייקטים
-   OkHttp Logging Interceptor - לוגים לבקשות רשת
-   Gson - עיבוד פורמט JSON

## 📊 גרפים ונתונים

-   MPAndroidChart - גרפי עוגה למסך הסטטיסטיקות

## 🎭 אנימציות

Lottie - מוכנות מראש JSON אנימציות  
משמש לאפקטים בממשק וsplash screen

## 🔄 Background Work

-   WorkManager - משימות רקע מתוזמנות
